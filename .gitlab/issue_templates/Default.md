### Before you file: is a node blank, broken, or missing its buttons after an update?

This is almost always a **browser cache**, not a bug. Please try the 30-second fix first:
[Read this first](https://gitlab.com/pixaroma/comfyui-pixaroma/-/issues/2)

Hard-refresh with **Ctrl + Shift + R** (Windows / Linux) or **Cmd + Shift + R** (Mac). If that fixes it, you do not need to file anything.

---

If it is still broken after a hard refresh (or this is a different bug), please fill in the details below and delete the section above:

**What is happening?**
Describe the problem. Screenshots help a lot.

**Steps to reproduce**
What did you do right before it happened?

**Versions** (please include)
Drop a **Version Check Pixaroma** node on the canvas, click its **Copy** button, and paste the result here.

**Console errors** (if a node is blank or broken)
Press **F12**, click **Console**, reload the page, then paste or screenshot any red error lines.
